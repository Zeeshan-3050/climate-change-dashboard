# Global Climate Change Dashboard

An interactive dashboard built in Google Colab to visualize and forecast key indicators of global climate change.

## 🚀 Project Overview
This project aggregates data from NASA, NOAA, and Our World in Data to provide insights into:
- **Temperature Anomalies:** Historical trends in global surface temperatures (1880-Present).
- **CO₂ Emissions:** Comparative analysis of annual emissions by major countries.
- **Socioeconomic Correlations:** Relationship between GDP, Population, and Environmental impact.
- **Geospatial Evolution:** Animated maps showing the spread of global CO₂ emissions since 1990.
- **Cryosphere & Oceans:** Tracking sea level rise and the declining mass of polar ice sheets.
- **Forecasting:** 20-year predictive modeling of temperature anomalies using the Facebook Prophet library.

## 🛠️ Technology Stack
- **Python** (Pandas, NumPy)
- **Plotly** (Interactive Visualizations)
- **Prophet** (Time-series Forecasting)
- **GitHub API** (Automated Deployment)
